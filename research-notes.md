# Research Notes - Vibe Coding

## Concept
Vibe Coding is an approach where the "vibe" or aesthetic of the code and the UI drives the development process. It focuses on high-quality design, smooth transitions, and a premium feel from the start.

## Observations
- The Futurense UI uses **Deep Navy/True Black** as a base.
- **Glassmorphism** (backdrop-filter: blur) is used for cards.
- **Pill-shaped** buttons and navigation elements are a core design language.
- **Typography** is clean and geometric (SF Pro Display / Inter).
- **Dual Theme** support is essential for a premium tech feel.
